# 解决无法点击按钮的情况

## 调整「游戏分辨率」「窗口化」

- **「分辨率」和「跑图按钮」清晰度有关系**
- **我的素材是「低分辨率」，所以「默认图片可能点击失败」**
- **确保游戏窗口是「窗口化」而不是「无边框」**
